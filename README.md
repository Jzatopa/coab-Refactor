# coab - Curse of the Azure Bonds
This is a reprogramming of the DOS game by the same name for modern Windows (via .NET). Coded in C#.

The game is almost 100% feature complete, besides player importation from Pool of Radiance or Hillsfar.

All the original game cheats plus more are available, just right click on the game window to enable/disable.

## High-resolution 2K version in development

Work is underway on a high-resolution 2K presentation for Curse of the Azure Bonds, targeting modern 2560x1440 displays while preserving the original 320x200 game logic and layout. Current work includes Linux/Mono support, enlarged title and selected BIGPIC artwork, an original-glyph HD font with an optional quill/brush alternative, cinematic audio replacements, and atomic combat-area presentation. Retained HD visuals follow the original framebuffer replacement boundaries documented in [HD_ASSET_LIFECYCLE.md](HD_ASSET_LIFECYCLE.md). The upgrade remains in progress and experimental assets may change as the refactor continues.

The repository `HDAssets/` tree is generated by `scripts/build_hd_asset_tree.py`. It mirrors the 21 folders in `original assets/`, uses the original filename plus `_HD.png`, and currently contains 126 verified counterparts. The isolated full-auto launcher validates the 231-entry identity ledger and atomically stages 118 approved assets into `runtime/full-auto/data/HDAssets/integrated`; the runtime catalog uses exact archive/block/frame lookup and the retained picture layer is cleared at panel, world, outer-frame, and combat replacement boundaries.
